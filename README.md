# 🐱 Toro Inoue Wallpaper Gallery

> My First Web Project! Built with ❤️ using Claude AI & ChatGPT

An aesthetic and interactive wallpaper gallery dedicated to **Toro Inoue**, a beloved minimalist character. This project combines beautiful design, smooth interactions, and cozy vibes into one delightful browsing experience.

## ✨ Features

### Gallery & Discovery
- **Pinterest-Style Masonry Layout** - Responsive grid that adapts to any screen size
- **Smart Search** - Find wallpapers by name instantly
- **Color Filter** - Discover wallpapers by aesthetic color palette (#A8D8FF, #FFB7C5)
- **Download & Save** - Download favorite wallpapers in high quality
- **Favorites System** - Mark your beloved wallpapers (persistent with LocalStorage)

### Ambiance & Vibes
- **Cozy Music Player** - 3 aesthetic playlists to browse by
  - "Relaxed" by MC Mablo Dos Paredões
  - "Can't Take My Eyes Off You" by Craymer, AIIVAWN
  - "Tired Boy" by Joey Pecoraro
- **Floating Emoji Animations** - Magical particles float across your screen ✨
- **Toro Decorations** - Cute Toro characters watching you browse
- **Dark Mode** - Eye-friendly browsing anytime (with persistent preference)

### User Experience
- **Smooth Animations** - Everything transitions beautifully
- **Responsive Design** - Perfect on desktop, tablet, and mobile
- **Preview Modal** - Click to see wallpapers in full detail
- **Volume Control** - Adjust music to your preference
- **Persistent Settings** - Your favorites, dark mode, and music preference save automatically

## 🛠️ Tech Stack

- **HTML5** - Semantic markup
- **CSS3** - Modern styling with animations & gradients
- **Vanilla JavaScript** - No frameworks, pure ES6+
- **LocalStorage API** - For persistent user preferences
- **JSON** - Scalable wallpaper database
